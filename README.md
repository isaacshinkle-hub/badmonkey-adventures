# Bad Monkey Adventures

Static GitHub Pages site for **Bad Monkey Adventures** — guided sailing and adventure motorcycle trips.

- Live: https://badmonkeyadventures.com/
- Repo: https://github.com/isaacshinkle-hub/badmonkey-adventures
- Contact: badmonkeyadventures@gmail.com

## Structure
- `index.html` — home
- `sailing.html` — Bad Monkey Sailing / SV Sea Monkey
- `moto.html` — Bad Monkey Moto
- `style.css` — shared styles
- `images/logos/` — brand stickers/logos
- `images/sailing/` — sailing photos used on site
- `images/moto/` — moto photos used on site
- `CNAME` — custom domain

Polished with real logos and trip photos. Trip calendar still placeholder/planning status.
